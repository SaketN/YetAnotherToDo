# Yet Another Todo
Just another implementation of TODO list in angularjs and Node. Store todos to mongodb.

--- Server Setup
pre-requisite : 
install monog npm install mongoClient
run mongodb ( mongod)

in /api do npm install 

do npm start to start the api server

--- client setup
in /app 
do bower install
install http server  npm install http-server -g

run http server http-server -p 3333
access app on http://localhost:3333
